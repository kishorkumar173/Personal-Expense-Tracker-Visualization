# 💸 Personal Expense Tracker Dashboard

A professional Streamlit-based Personal Expense Tracker Dashboard that helps users track, analyze, and visualize expenses with interactive charts, KPI metrics, CSV upload support, and PDF report generation.

---

# 📌 Project Overview

Managing personal finances manually can be difficult and time-consuming. This project provides a smart dashboard that allows users to:

- Track expenses
- Analyze spending habits
- Visualize financial trends
- Generate downloadable reports
- Understand category-wise spending

This project simulates a real-world personal finance analytics system using Python and Streamlit.
